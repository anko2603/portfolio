## Packages
framer-motion | Essential for scroll-triggered animations and the scrollytelling canvas
lucide-react | Beautiful icons for UI elements
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility for merging Tailwind classes safely

## Notes
- The scrollytelling canvas expects a sequence of images at `/sequence/frame_[index].webp`.
- If images are missing, the canvas will render a fallback visual (expanding circle/progress).
- Fonts used: 'Space Grotesk' for display/headings, 'Inter' for body.
- Dark mode default with #121212 background.
